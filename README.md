# prog-lang 编程语言与框架入门学习代码和笔记

## 介绍
golang 和 rust 是近年来讨论热度比较高的两门编程语言，go 主要用于云原生开发，rust 主要用于系统编程。
这两门语言都有很多优秀的特性，但学习起来也有一定的难度。
本仓库主要记录我学习这两门语言的代码例程和笔记，也可能会增加一些其他编程语言的学习笔记。
